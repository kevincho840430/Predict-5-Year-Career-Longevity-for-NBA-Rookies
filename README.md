# Predict-5-Year-Career-Longevity-for-NBA-Rookies

# 說明
每年，來自世界各地的數百萬籃球迷都收看NBA選秀，希望他們最喜歡的球隊獲得總冠軍，並發現下一位偉大的NBA球星。

這些NBA球隊的工作人員花費數千小時來搜尋和評估大學和國際人才，以尋找能夠在職業水平上取得成功並為球隊做出貢獻的球員。

隨著數據科學領域的發展，嘗試和評估超越傳統方法的人才是有意義的。
